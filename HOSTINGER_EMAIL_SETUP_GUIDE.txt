🔧 HOSTINGER EMAIL SETUP GUIDE
================================

❌ PROBLEM: Emails nahi ja rahe hain
✅ SOLUTION: Hostinger mein email account banana padega

📋 STEP-BY-STEP SETUP:
======================

STEP 1: Create Email Account in Hostinger
------------------------------------------
1. Login to Hostinger hPanel
2. Go to "Emails" section
3. Click "Create Email Account"
4. Create email: admin-orders@boultindia.com
5. Set a strong password (save it!)
6. Click "Create"

STEP 2: Update PHP Script
--------------------------
1. Open send-order-email-smtp.php
2. Find line: $smtp_password = 'YOUR_EMAIL_PASSWORD_HERE';
3. Replace with actual password from Step 1
4. Save file

STEP 3: Upload Updated PHP File
--------------------------------
1. Upload send-order-email-smtp.php to Hostinger
2. Rename it to: send-order-email.php
3. Location: public_html/send-order-email.php
4. Replace existing file

STEP 4: Test Email
------------------
1. Run: node test-hostinger-email.js
2. Check customer email inbox
3. Check vtechmultisolutions@gmail.com inbox
