## 🚀 Hostinger Upload Instructions

### Step 1: Upload E-commerce
1. Go to Hostinger File Manager
2. Navigate to `public_html/`
3. Upload all files from `boult-react-ecommerce/build/`
4. Ensure `.htaccess` is uploaded

### Step 2: Upload Admin Panel
1. Create folder `public_html/admin/`
2. Upload all files from `boult-react-admin/build/`
3. Ensure `.htaccess` is uploaded

### Step 3: Verify HTTPS
- Ensure SSL certificate is active
- Force HTTPS redirect

## 🔧 Razorpay Configuration

### Required Steps:
1. **Login to Razorpay Dashboard**
2. **Go to Settings > Website and App URLs**
3. **Add your domain**:
   ```
   https://your-domain.com
   https://www.your-domain.com
   ```
4. **Save settings**

### Test Payment:
- Add product to cart
- Go to checkout
- Select card/UPI payment
- Verify modal opens
- Complete test transaction
